### Render Deployment
- [ ] Go to [dashboard.render.com](https://dashboard.render.com)
- [ ] Click "New +" → "Blueprint"
- [ ] Connect GitHub repository

## 🆘 Troubleshooting

### Build Fails
**Problem:** "Torch package too large"
**Solution:** The ML dependencies are large. Options:
1. Use CPU-only torch (smaller)
2. Upgrade to Render paid tier ($7/mo for more resources)
3. Remove transformers/torch from requirements temporarily

### App Doesn't Start
**Problem:** App crashes on startup
**Solution:**
1. Check Render logs: Dashboard → Service → Logs
2. Verify FINNHUB_API_KEY is set correctly
3. Test health endpoint: `curl https://your-app.com/health`

### Can't Log In
**Problem:** Database connection error
**Solution:**
1. For SQLite: Ensure `/app/instance` directory exists
2. For PostgreSQL: Verify DATABASE_URL format
3. Check SECRET_KEY is set (auto-generated by Render)

### Slow Response
**Problem:** First request after idle is slow
**Solution:**
- Free tier sleeps after 15 min inactivity
- First request wakes it up (takes ~30 seconds)
- Upgrade to paid tier for always-on ($7/mo)
