Using Expected Moves for Swing Trades

  • • Stop Loss Placement: Set stops outside the expected daily move to avoid noise
  • • Profit Targets: Use expected weekly move as minimum profit target
  • • High Vol: Wider stops needed, smaller positions (reduce by 30-50%)
  • • Low Vol: Tighter stops work well, watch for volatility expansion
  • • Position Sizing: In high vol, risk same $ amount but with fewer shares
